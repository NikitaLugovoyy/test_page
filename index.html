Основные типы узлов в Corezoid

Corezoid предоставляет различные типы узлов для построения бизнес-процессов. Каждый узел выполняет определенную функцию и имеет свои особенности конфигурации.

1. Copy Task

Узел Copy Task используется для копирования и модификации данных процесса. Позволяет добавлять, изменять или удалять параметры.

{ "name": "John Doe", "email": "john@example.com", "status": "new_user" }

2. API Call

Узел для выполнения HTTP запросов к внешним API. Поддерживает различные методы (GET, POST, PUT, DELETE) и форматы данных.

3. Condition

Логический узел для создания ветвлений в процессе. Позволяет направлять поток выполнения по разным путям в зависимости от условий.

4. Timer

Узел для создания временных задержек в процессе. Полезен для реализации timeout'ов или отложенных действий.
